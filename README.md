# Prueba Técnica SQL

Este repositorio contiene la resolución de una prueba técnica de SQL.

##  Descripción

El script `Prueba tecnica Emmanuel Oswaldo Rizo para TOKA.sql` crea una base de datos desde cero, inserta los datos iniciales y resuelve una serie de ejercicios prácticos.

## Habilidades Demostradas en la prueba

* **DDL (Data Definition Language):**
    * `CREATE DATABASE` y `CREATE TABLE`
    * Definición de `PRIMARY KEY` y `FOREIGN KEY`
    * `ALTER TABLE` para añadir nuevas columnas.
* **DML (Data Manipulation Language):**
    * `INSERT INTO` para poblar las tablas.
    * `SELECT` con `JOIN` para unir múltiples tablas.
    * `UPDATE` para modificar datos existentes.
    * `DELETE` usando la llave primaria para borrado seguro.
* **Técnicas Avanzadas:**
    * Uso de `UPDATE` con `CASE` para actualizaciones masivas eficientes.
    * Verificación de cambios mediante `SELECT` después de cada modificación.
